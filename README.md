# Rock Paper Scissors
 
A browser-based Rock Paper Scissors game built with vanilla HTML, CSS and JavaScript.
 
**Live:** [http://89.167.40.206](http://89.167.40.206)

![Rock Paper Scissors](assets/Screenshot.png)
 
## Features
 
- Play against the computer
- Score tracking (Wins, Losses, Ties)
- Score persists across sessions via `localStorage`
- Hosted by me
## Tech Stack
 
`HTML` `CSS` `JavaScript`
